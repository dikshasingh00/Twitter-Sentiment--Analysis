# Twitter-Sentiment--Analysis
Real-Time Twitter Sentiment Analysis using NLP 
## Project Overview
This project performs Twitter sentiment analysis using Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- NLP
- VADER Sentiment Analysis
- Matplotlib
- WordCloud
- Streamlit

## Features
- Text cleaning
- Sentiment classification
- Positive, Negative and Neutral analysis
- Data visualization
- Interactive dashboard

## Output
The system analyzes tweets and displays sentiment results with graphs.
